# codex-multi-agent 한국어 안내

[영문 기준 README](README.md) | [한국어 가이드 인덱스](docs/ko/README.md)

이 저장소는 최종 제품 저장소가 아니라, 새 프로젝트를 안전하게 시작하기 위한 운영 템플릿이자 기반 저장소입니다.

핵심 역할은 다음과 같습니다.

- Codex app: 사람이 작업을 지시하고 결과를 검토하며 승인하는 주 인터페이스입니다.
- Codex CLI: 계획, 실행, 리뷰 작업을 실제로 돌리는 작업 엔진입니다.
- 사람(supervisor): 무엇을 할지 정하고, 결과를 읽고, 실행을 승인하거나 거절합니다.

중요한 점:

- 영어 문서와 내부 설정 파일이 운영 기준 문서입니다.
- 한국어 문서는 실제 사용자를 위한 학습용, 안내용 레이어입니다.
- 첫 읽기 전용 계획 실행은 프로젝트 skill catalog를 먼저 확인하고, 어떤 skill이 어떤 하위 작업에 맞는지 routing 결과를 아티팩트에 남깁니다.
- 이 저장소는 웹 전용 템플릿이 아닙니다. 웹 앱, 데스크톱 앱, 백엔드, 채팅 시스템, 시뮬레이션, 게임, 하이브리드 시스템 등에 모두 적용할 수 있습니다.

## 가장 먼저 할 일

실제 프로젝트에 사용할 때는 이 저장소를 바로 쓰기보다 먼저 포크해서 자신의 프로젝트용 저장소로 시작하는 것을 권장합니다.

그다음 순서:

1. 이 저장소를 포크합니다.
2. 자신의 포크를 클론합니다.
3. Codex app에서 엽니다.
4. 읽기 전용 계획부터 시작합니다.
