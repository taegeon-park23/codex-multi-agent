# Automation Ideas

Potential future automations after the product stack is chosen:

- daily repository hygiene check
- template drift check for README and AGENTS
- review readiness check for active tasks
- stale worktree reminder
- container build smoke check on demand

Do not automate these yet:
- stack selection
- architecture approval
- dependency approval
- destructive actions

Keep automations human-in-the-loop and scoped to clear outputs.
